# Semantic Search over Dataset Descriptions

This project aims to build a semantic search system over dataset descriptions using natural language processing (NLP) techniques. The goal is to improve dataset discovery by going beyond keyword-based search and enabling meaning-based retrieval.
Developed as part of a bachelor's thesis.

---
## Project Overview
Traditional search methods (e.g., keyword matching) often fail to capture the semantic meaning of user queries. In this project, we design and implement a pipeline that:

- Collects dataset descriptions from online sources (e.g., Kaggle, Hugging Face)
- Cleans and normalizes textual data
- Builds a structured dataset
- Enables semantic retrieval using vector representations

---

## Pipeline
The current pipeline consists of the following steps:

1. **Data Collection**
   - `collect_kaggle_refs.py`
   - `fetch_kaggle_descriptions.py`
   - `collect_hf.py`

2. **Data Cleaning & Preprocessing**
   - `clean_kaggle.py`
   - `clean_hf.py`
   - `merge_all.py`
   - `normalize_merge.py`

3. **Indexing (in progress)**
   - `build_faiss_index.py`
